Viewport Meta Tag

El punto de partida obligatorio de todo diseño responsivo

¿Qué hace esta meta tag?

Sin el <meta name="viewport">, los navegadores móviles simulan un viewport de 980px y reducen toda la página con zoom-out. Con ella, se le indica al navegador que use el ancho real del dispositivo como referencia y no aplique ningún zoom inicial. 

<meta name="viewport" content="width=device-width, initial-scale=1">
Con viewport meta

El contenido ocupa el 100% del ancho del dispositivo sin zoom artificial.

100% del ancho real
width device-width
initial-scale 1
Sin viewport meta

El navegador simula 980px y aplica zoom-out. El contenido se ve muy pequeño.

~35% del ancho real (zoom out)
viewport simulado 980px

Parámetros del atributo content

width=device-width

Usa el ancho físico del dispositivo como ancho del viewport.

initial-scale=1

No aplica zoom inicial. La página se muestra a escala 1:1.

minimum-scale=1

Evita que el usuario haga zoom-out más allá de la escala base.

user-scalable=no

⚠ Evitar: bloquea el zoom del usuario, violando WCAG 1.4.4 (Redimensionamiento de texto).

Regla práctica

Incluye siempre width=device-width, initial-scale=1 en el <head> antes de cualquier hoja de estilos. Sin esta línea, las media queries con max-width o min-width no se evaluarán contra el viewport real del dispositivo.
