Media Queries

Aplica estilos condicionalmente según las características del dispositivo

¿Qué son las media queries?

Una @media query envuelve un bloque de CSS que solo se aplica cuando se cumple una condición sobre el medio: ancho del viewport, orientación, resolución de pantalla o preferencias del sistema operativo. 

@media (min-width: 768px) {
  /* estos estilos solo aplican cuando el viewport es ≥ 768px */
  .sidebar { display: block; }
}

Otros tipos de media queries

@media (orientation: landscape)

Se aplica cuando el dispositivo está en modo horizontal.

@media (prefers-color-scheme: dark)

Se activa cuando el usuario tiene el modo oscuro del sistema activado.

@media (prefers-reduced-motion: reduce)

Respeta la preferencia de reducir animaciones. Clave para WCAG 2.3.3.

@media print

Estilos que solo aplican al imprimir la página.

Código completo del demo

/* Base: móvil — una sola columna */
.responsive-layout {
  display: flex;
  flex-direction: column;
  gap: 1rem;
}

.layout-sidebar {
  display: none; /* oculto en móvil */
}

/* Tablet: mostrar sidebar en fila */
@media (min-width: 768px) {
  .responsive-layout {
    flex-direction: row;
  }
  .layout-sidebar {
    display: block;
    width: 200px;
    flex-shrink: 0;
  }
}

/* Escritorio: sidebar más ancha */
@media (min-width: 1024px) {
  .layout-sidebar {
    width: 260px;
  }

  .content-block {
    flex: 1 0 200px;
  }
}
@media (min-width: 1280px) {
  .layout-sidebar {
    width: 260px;
  }
}
