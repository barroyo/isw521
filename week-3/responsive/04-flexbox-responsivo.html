Flexbox Responsivo

Distribuye y redistribuye elementos con flex-wrap y valores flex

La clave: flex-wrap y flex shorthand

Flexbox se vuelve responsivo combinando flex-wrap: wrap (permite salto de línea) con el shorthand flex: grow shrink basis. El flex-basis actúa como un breakpoint implícito: cuando el elemento no cabe en su base, salta a una nueva línea. 

.contenedor {
  display: flex;
  flex-wrap: wrap;   /* permite que los items salten de línea */
  gap: 1rem;
}

.item {
  flex: 1 1 250px;
  /*   ↑ ↑ ↑
       │ │ └── basis: ancho base deseado
       │ └──── shrink: puede encogerse
       └────── grow: puede crecer */
}

Propiedades esenciales

flex-wrap: wrap

Permite que los ítems salten a la siguiente línea cuando no caben.

flex: 1 1 250px

El ítem crece, se encoge y tiene una base de 250px. Shorthand de grow, shrink y basis.

gap: 1rem

Espacio entre ítems tanto en filas como en columnas. Reemplaza márgenes manuales.

min-width: 0

Evita que el contenido interno desborde cuando el ítem se encoge.

align-items: stretch

Por defecto, todos los ítems en una fila tienen la misma altura.

flex-direction

Cambia entre row (horizontal) y column (vertical).
