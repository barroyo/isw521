CSS Grid Responsivo

Columnas automáticas con auto-fit, minmax y template-areas

auto-fit + minmax: columnas sin media queries

La combinación repeat(auto-fit, minmax(min, 1fr)) crea tantas columnas como quepan en el contenedor. auto-fit colapsa las columnas vacías para que los ítems se estiren; auto-fill las mantiene reservadas. 

.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 1.5rem;
}
/* Sin una sola @media query:
   viewport 400px  → 1 columna
   viewport 650px  → 2 columnas  (2 × ~200px + gap)
   viewport 900px  → 3 columnas
   viewport 1200px → 4+ columnas */

    "header  header"
    "nav     nav"
    "main    sidebar"
    "footer  footer";
  grid-template-columns: 1fr 220px;

/* Móvil */
@media (max-width: 680px) {
  .page-areas {
    grid-template-areas:
      "header"
      "nav"
      "main"
      "sidebar"
      "footer";
    grid-template-columns: 1fr;
  }
}
