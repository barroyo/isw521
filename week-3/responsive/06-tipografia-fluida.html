Tipografía Fluida

Escala continua con clamp() sin puntos de quiebre artificiales

La función clamp()

clamp(mínimo, valor-ideal, máximo) devuelve el valor ideal mientras esté entre el mínimo y el máximo. Combinando el valor ideal con unidades relativas al viewport (vw), el resultado escala fluidamente según el ancho de la pantalla. 

font-size: clamp(1rem, 2.5vw, 2rem);
/*              ↑         ↑       ↑
                mínimo  ideal   máximo
                16px    fluido  32px  */

Título principal fluido

Subtítulo fluido de sección

Este párrafo usa clamp() para escalar su tamaño de fuente y su interlineado de forma continua. No hay saltos abruptos entre tamaños — la transición es suave en cualquier viewport entre móvil y escritorio amplio.

Espaciado fluido

clamp() no solo aplica a tipografía. También funciona para padding, margin y cualquier propiedad de longitud. 

.section {
  /* padding que escala de 1rem a 4rem */
  padding: clamp(1rem, 5vw, 4rem);
}

.container {
  /* ancho máximo centrado con padding fluido */
  max-width: 1200px;
  margin-inline: auto;
  padding-inline: clamp(1rem, 4vw, 3rem);
}

.card {
  /* gap fluido en grid/flex */
  gap: clamp(0.75rem, 2vw, 1.5rem);
}

clamp() vs múltiples @media

Antes — múltiples media queries
h1 { font-size: 1.5rem; }

@media (min-width: 480px) {
  h1 { font-size: 2rem; }
}
@media (min-width: 768px) {
  h1 { font-size: 2.5rem; }
}
@media (min-width: 1024px) {
  h1 { font-size: 3rem; }
}
Ahora — una sola línea
h1 {
  font-size: clamp(1.5rem, 4vw, 3rem);
}

/* Escala continua y suave
   entre 1.5rem y 3rem,
   sin saltos visibles */
