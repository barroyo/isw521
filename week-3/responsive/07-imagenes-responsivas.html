Imágenes Responsivas

max-width, object-fit, aspect-ratio y srcset para imágenes que se adaptan

El mínimo imprescindible

Sin ningún CSS, una imagen conserva su tamaño intrínseco y puede desbordar su contenedor en pantallas pequeñas. La regla base más importante del diseño responsivo para imágenes es: 

img {
  max-width: 100%;  /* nunca más ancha que su contenedor */
  height: auto;     /* mantiene la proporción original */
  display: block;   /* elimina el espacio inline por defecto */
}

Técnicas CSS para imágenes

1 Imagen fluida básica
max-width: 100%
img {
  max-width: 100%;
  height: auto;
}

Se encoge si el contenedor encoge. Nunca excede el ancho del contenedor.

2 object-fit: cover
object-fit: cover
.card-img {
  width: 100%;
  height: 200px;
  object-fit: cover;
  object-position: center top;
}

Rellena el contenedor recortando la imagen sin deformarla. Ideal para tarjetas con altura fija.

3 aspect-ratio
aspect-ratio: 16 / 9
.hero-img {
  width: 100%;
  aspect-ratio: 16 / 9;
  object-fit: cover;
  /* Sin height fijo:
     la altura se calcula
     automáticamente */
}

Mantiene la proporción sin definir una altura fija. Escala fluidamente con el ancho.

4 object-fit: contain
object-fit: contain
.logo-img {
  width: 100%;
  height: 120px;
  object-fit: contain;
  /* Muestra toda la imagen
     sin recortar,
     puede dejar espacio
     en los lados */
}

Muestra la imagen completa dentro del contenedor sin recortar. Útil para logos y diagramas.

srcset y sizes: imágenes según resolución

El atributo HTML srcset permite al navegador elegir la versión de la imagen más apropiada según el ancho del viewport y la densidad de píxeles, ahorrando ancho de banda en móviles. 

<img
  srcset="imagen-400.jpg 400w,
          imagen-800.jpg 800w,
          imagen-1200.jpg 1200w"
  sizes="(max-width: 480px) 100vw,
         (max-width: 1024px) 50vw,
         33vw"
  src="imagen-800.jpg"
  alt="Descripción de la imagen"
  loading="lazy"
>

<!-- picture: imágenes artísticas o formatos distintos -->
<picture>
  <source media="(min-width: 768px)" srcset="imagen-wide.webp">
  <source media="(max-width: 767px)" srcset="imagen-crop.webp">
  <img src="imagen-fallback.jpg" alt="Descripción">
</picture>
